# Веб-сервис "Ассистент по языку запросов 1С:Предприятие 8"

## Описание

Веб-сервис разработан для помощи продвинутым аналитикам в создании запросов на
платформе 1С:Предприятие 8.

Решаемые задачи:
 - Интеграция, позволяющая работать пользователям из привычного интерфейса
   1С:Предприятие 8.
 - Предоставление ИИ-агенту контекста для создания текста запросов: список и
   описание таблиц информационной базы.
 - Уведомление ИИ-агента о результатах проверки синтаксиса запроса.

Интеллектальным агентом в текущей версии выступает модель **GigaChat**.
 
Доступна работа в двух режимах:
 - **Режим веб-сервиса**. Подразумевается интеграция с внешней обработкой
   "Консоль запросов" или любым подобным инструментом. Рекомендуемый режим.
 - **Режим консоли**. Работа выполняется в командной строке. Режим удобен
   для тестирования и отладки. Также в этом режиме доступно обновление базы
   данных с описанием таблиц информационной базы.
